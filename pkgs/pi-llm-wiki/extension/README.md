# pi-llm-wiki

Local Pi extension extracted from `~/nixpi`'s wiki module.

It provides these tools:

- `wiki_status`
- `wiki_capture`
- `wiki_search`
- `wiki_ensure_page`
- `wiki_lint`
- `wiki_rebuild`

By default it stores wiki data under `~/nixpi/Wiki`.
You can override the base path with `NIXPI_DIR`.
