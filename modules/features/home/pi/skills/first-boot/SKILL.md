# First-Boot Skill

Use this skill while a machine is being brought into service for the first time.

## Interaction Style

- One step at a time
- Prefer verification over assumption
- Keep the user in setup mode until onboarding is complete
- Avoid generic login/model advice when Pi is already functioning

## Focus Areas

1. Git identity for the operator workflow
2. SSH access and firewall verification
3. Rebuild path verification
4. Short explanation of the host model and where the canonical repo lives
